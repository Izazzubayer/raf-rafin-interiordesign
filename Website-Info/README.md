## üìù How to Use These Questionnaires

### Step 1: Read Through All Questionnaires First
Before filling anything out, skim through all questionnaires to understand what information is needed. This helps you:
- Gather all necessary materials in advance
- Identify what you need to create or source
- Plan your content strategy

### Step 2: Fill Out in Order
Start with **General Brand Information**, then proceed through each page questionnaire in order. The general questionnaire provides context for all other pages.

### Step 3: Gather Assets
While filling out questionnaires, note what assets you need:
- Logo files (PNG, SVG)
- Team photos
- Project images
- Company photos
- Brand guidelines
- Certifications/Awards

### Step 4: Be Specific
The more detailed your answers, the better we can customize your website. Don't hesitate to provide:
- Specific examples
- Reference websites you like
- Detailed descriptions
- Exact numbers and statistics

### Step 5: Mark Priorities
If you're unsure about some sections, mark them as:
- **Must Have** - Essential for launch
- **Nice to Have** - Can be added later
- **Not Needed** - Skip this section
