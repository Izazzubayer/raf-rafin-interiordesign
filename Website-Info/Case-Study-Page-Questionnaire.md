# Case Study Page Content & Functionality Questionnaire

This document contains questions about the content and functionality for your case study pages (detailed project pages). These pages should feel premium, editorial, and emotionally engaging while showcasing your design process and results. Please provide detailed answers to help us create compelling case studies.

### 1. CASE STUDY APPROACH

#### Questions:
1. **Case Study vs. Standard Project Page**:
   - Should ALL projects have full case study pages, or only selected/featured projects?
   - Should there be two tiers: "Standard Project Pages" and "Detailed Case Studies"?
   - What criteria determines if a project gets a full case study? (e.g., Budget, Complexity, Awards, Client Permission)

2. **Case Study Format**:
